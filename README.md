Digital Timer

Description
The Digital Timer is a web-based application that displays the current date and time. It updates every second to show the day of the week, date, month, year, hours, minutes, seconds, and AM/PM notation in a user-friendly 12-hour format.

Features
- Displays the current day of the week
- Displays the current date with leading zeros if necessary
- Displays the current month (e.g., Jan, Feb)
- Displays the current year
- Displays the current time in 12-hour format with AM/PM notation
- Updates every second automatically

Installation
- Clone the repository or download the files.
  https://github.com/Nkiru-Ujam/Digital-Timer.git
- Navigate to the project directory
  cd Digital-Timer
- Open index.html in your web browser to view the digital timer.

Usage
Simply open the index.html file in a web browser.
The digital timer will start automatically and update every second.
